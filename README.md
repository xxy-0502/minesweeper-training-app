# 扫雷康复训练游戏系统

课程设计前端代码区，使用 Vite + React + TypeScript 初始化。

## 技术栈

- Vite
- React
- TypeScript
- React Router
- Zustand
- ECharts
- Oxlint + Prettier

## 常用命令

```bash
pnpm install
pnpm dev
pnpm build
pnpm lint
pnpm format
```

## 当前目录

```text
src/
  data/          难度、时间等静态配置
  pages/         配置页、游戏页、报告页
  store/         训练配置状态
  types/         业务类型定义
```
